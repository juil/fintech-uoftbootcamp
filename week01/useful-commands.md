# Useful Commands

## Terminal

### Better Tools

- Editor: `vim` -> `nvim` [NeoVim](https://neovim.io/)
- Reader: `cat`
	- `less` Output without cluttering terminal buffer.
	- `mdcat`, `glow` Markdown output.

### Navigation

#### PATH

See path of program:

	type type
	which which

#### Alias

Edit `~/.aliases`

	alias shortcut='cd ~/path'
	alias cmd='type command`
