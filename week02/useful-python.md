### Iterate through dictionary

```
>>> a_dict = {'color': 'blue', 'fruit': 'apple', 'pet': 'dog'}
>>> for key in a_dict:
...     print(key, '->', a_dict[key])
...
```
color -> blue
fruit -> apple
pet -> dog
